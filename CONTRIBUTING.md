# Contribution Guidelines

Thank you for your interest in contributing to the AI Investment Collection project!

## How to Contribute
1. **Fork the Repository**: Use the GitHub interface to create a copy of this repository in your own account.
2. **Clone Your Fork**: Clone your forked version locally to make changes.
3. **Create a New Branch**: It's best practice to create a new branch for each feature or fix you are working on. Use a descriptive name for your branch.
4. **Make Your Changes**: Implement the changes you'd like to contribute.
5. **Write Tests**: Ensure that your changes are covered by tests (if applicable).
6. **Commit Your Changes**: Make commits that reflect the purpose of your changes. Write clear and concise commit messages.
7. **Push Your Changes**: Push your changes to your fork on GitHub.
8. **Open a Pull Request**: Go to the original repository where you want to contribute and create a pull request from your branch.

## Code of Conduct
Please adhere to our [Code of Conduct](CODE_OF_CONDUCT.md) when contributing to this project.

## Questions?
If you have any questions about the contributing process, feel free to open an issue in the repository. We'll be happy to assist you!
